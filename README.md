# 🔍 Buscador de RFC en Archivos XML

Este proyecto permite buscar un RFC dentro de grandes cantidades de archivos XML almacenados en una carpeta del equipo.
El sistema recorre automáticamente la carpeta seleccionada y todas sus subcarpetas para encontrar coincidencias.

---

## 🚀 Funcionalidades

* Búsqueda de RFC en múltiples archivos XML
* Exploración automática de subcarpetas
* Muestra información relevante de cada coincidencia:

  * 📄 Nombre del archivo
  * 📁 Ruta completa
  * 📅 Fecha de creación
  * 💰 Monto

---

## 🛠️ Requisitos

* Python instalado
* Librerías incluidas en `requirements.txt`

---

## ▶️ Ejecución

1. Ejecutar el archivo:

   ```bash
   python Buscador_Tarea2.py
   ```

2. Abrir el navegador y entrar a:

   ```
   http://127.0.0.1:5000
   ```

3. Ingresar:

   * La ruta de la carpeta a analizar
   * El RFC a buscar

---

## 📌 Uso

El sistema analizará todos los archivos XML dentro de la carpeta indicada (incluyendo subcarpetas) y mostrará cada coincidencia encontrada junto con su información.
